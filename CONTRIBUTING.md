# Contribuir

1. Fork → clona **fuera** del vault → `npm install` → `npm run build`.
2. Un tema por PR; incluye pasos para reproducir en issues.
3. No subas bases `.db` ni datos personales.

Para probar la integración con tareas, instala también [vault-task-board](https://github.com/FernandoRP-nan/vault-task-board).
